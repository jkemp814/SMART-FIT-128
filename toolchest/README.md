# Toolchest

This folder is a general-purpose storage area for reusable tools, scripts, and configuration files that support your Fedora development environment.

## Usage

- Store formatters, linters, utility scripts, and helper tools here for use across multiple projects or   - outside the devcontainer context.
- Examples: shellcheck, shfmt, custom Python/Node/Rust tools, backup scripts, or any workflow helpers.
- Use this folder for anything you want to keep handy but not tightly coupled to the devcontainer build
- process.

Feel free to organize, add, or remove tools as your workflow evolves.
